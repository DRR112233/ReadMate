# 心动书架 - 恋人共读 (Xindong Bookshelf)

一款专为情侣/异地恋设计的治愈系共读应用。在这里，阅读不再是孤独的旅程，而是与 TA 的心灵共鸣。

## ✨ 核心功能
- **📖 沉浸式阅读**：支持 EPUB/PDF 格式导入，精美的排版与多主题切换。
- **❤️ AI 恋人伴读**：内置智能 AI 伴侣，会根据你的阅读进度和感悟，实时留下深情的批注与回复。
- **📝 情绪手帐**：记录阅读时的点滴心情，支持编辑与永久保存，打造专属你们的文字记忆。
- **💬 实时互动**：支持划线分享，一键将触动心弦的句子发送给 TA 开启对话。
- **📱 全平台覆盖**：支持 PWA 安装及 Android 原生 App 体验。

## 🛠️ 技术栈
- **Frontend**: React 19 + TypeScript + Tailwind CSS
- **Animation**: Framer Motion
- **AI Engine**: Google Gemini API
- **Mobile Framework**: Capacitor (Android Support)
- **Parsing**: Epub.js + PDF.js

## 🚀 快速开始
1. 克隆本项目到本地。
2. 运行 `npm install` 安装依赖。
3. 运行 `npm run dev` 启动开发环境。
4. 运行 `npm run static-build` 生成生产环境包。

## 🎓 毕业设计说明
本项目旨在探索 AI 交互与数字阅读的结合，通过情感化的 UI 设计与智能化的反馈机制，提升用户的阅读粘性与情感体验。
